# closeEditor()

### Description

@short: Closes an editor

### Usage

~~~jsx {}
closeEditor(): void;
~~~

### Example

~~~jsx {6}
// create Scheduler
const calendar = new scheduler.Scheduler("#root", {
	// configuration parameters
});
// close an editor
calendar.closeEditor();
~~~
