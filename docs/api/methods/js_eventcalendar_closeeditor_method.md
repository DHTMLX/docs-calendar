# closeEditor()

### Description

@short: Closes an editor

### Usage

~~~jsx {}
closeEditor(): void;
~~~

### Example

~~~jsx {6}
// create Event Calendar
const calendar = new eventCalendar.EventCalendar("#root", {
	// configuration parameters
});
// close an editor
calendar.closeEditor();
~~~
